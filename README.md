# TangerineAuction

### Стек: 
- c# (.net 10)
- React
- Postgres
- Rabbit mq
- Redis
- Keycloak
- Minio

### Проекты:
- TangerineAuction.Web - веб апи + фронт
- TangerineGenerator.Service - микросервис для генерации мандаринок

### TangerineGenerator.Service
Генерирует редкость, название, цену и картинку (саму картинку кладёт в minio). Название, цена и картинка зависят от редкости. 
Картинка может содержать следующие элементы:
- Задний фон,
- Звёзды,
- Тень,
- Мандарин,
- Точки на мандарине,
- Блик,
- Ореол,
- Листик.

### Интеграция:
#### Mass transit:
- Содержит Request для запуска генерации мандаринки и возвращает её данные.
#### GRPC: 
- Получить версию сервиса;
- Получить временные ссылки на картинки из хранилища;
- Удалить картинку из хранилища.

### TangerineAuction.Web:
- Работа с сущностями (аукционы, мандарины, ставки);
- Работа с Keycloak (получение токена для confidential-client, создание системного пользователя);
- Запуск Hangfire Jobs (генерация мандаринок каждые 5 минут и очистка мандаринок каждый день);
- Связь с фронтом о новой ставке, новом аукционе и завершении аукциона через SignalR;
- Отправка сообщений на почту.

#### Интеграция:
- Делает паблиш Request-а для запуска генерации мандаринки.

### Алгоритм работы:
1. Срабатывает джоба в TangerineAuction.Web.
2. TangerineGenerator.Service генерирует мандаринку и отдаёт результат по rabbit mq.
3. TangerineAuction.Web обрабатывает запрос:
- создаёт аукцион;
- создаёт ставку равную изначальной стоимости мандаринки, автор ставки - системный пользователь;
- всё сохраняет в БД;
- оповещает фронт через SignalR о новом аукционе.

Раз в день TangerineAuction.Web проходит активным аукционам и проверяет кол-во ставок. Если ставок > 1, то отправляет чек победителю на почту и переводит аукцион в статус завершённого.
Если ставка только одна, то удаляет аукцион, мандаринку, ставки и фото мандаринки (по GRPC вызывает соответствующий метод у TangerineGenerator.Service).

Если пользователь перебил ставку, то отправляет письмо на почту пользователю, у которого перебили ставку.
